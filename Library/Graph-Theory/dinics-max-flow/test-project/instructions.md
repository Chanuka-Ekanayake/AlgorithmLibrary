# User Guide: Global CDN Bandwidth Router (Dinic's Algorithm)

This project simulates the network routing backend of a massive e-commerce platform. It models a Content Delivery Network (CDN) and uses **Dinic's Algorithm** to calculate the absolute maximum data flow from your central datacenter to a buyer's machine.

## How to Test

1. **Navigate** to the `test-project` folder.
2. **Run** the simulator:
   ```bash
   python app.py
   ```
