## How to Run

No external dependencies required — pure Python standard library.

### From the `tarjans-scc` root directory:

```bash
cd test-project
python app.py
```

### Or from the repo root:

```bash
python Library/Graph-Theory/tarjans-scc/test-project/app.py
```

### Expected Output:
Five demo sections, each solving a real-world SCC problem:
1. Microservice circular dependency detection
2. Social network mutual-influence communities
3. Compiler recursive function cycle detection
4. Fully strongly connected graph
5. Pure DAG (no cycles) pipeline
