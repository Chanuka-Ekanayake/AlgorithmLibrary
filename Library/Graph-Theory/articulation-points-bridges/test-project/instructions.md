# Running the Articulation Points & Bridges Test Project

## Quick Start
```bash
cd Library/Graph-Theory/articulation-points-bridges/test-project
python app.py
```

## What This Demo Shows
The application demonstrates 5 real-world applications of finding articulation points and bridges:
1. **Computer Network** - Critical router and connection analysis
2. **Transportation Network** - Critical intersection and road identification
3. **Social Network** - Key influencer detection
4. **Power Grid** - Infrastructure vulnerability assessment
5. **Simple Example** - Visual algorithm demonstration

## Expected Output
- Critical routers/intersections that would partition networks if removed
- Critical connections that would isolate network segments
- Vulnerability statistics and percentages
- Infrastructure hardening recommendations
- Visual graph examples with explanations

## No Additional Setup Required
All examples use built-in Python libraries and the articulation points implementation.
