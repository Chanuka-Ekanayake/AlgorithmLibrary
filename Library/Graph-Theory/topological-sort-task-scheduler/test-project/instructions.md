# Running the Topological Sort Test Project

## Quick Start
```bash
cd Library/Graph-Theory/topological-sort-task-scheduler/test-project
python app.py
```

## What This Demo Shows
The application demonstrates 5 real-world use cases of topological sort:
1. **Build System** - File compilation order
2. **Course Planner** - University course scheduling
3. **Package Manager** - Software installation order
4. **Project Management** - Task scheduling timeline
5. **Circular Dependency Detection** - Error handling

## Expected Output
- Build order for source files
- Semester-by-semester course plan
- Package installation sequence
- Project task timeline
- Cycle detection demonstration

## No Additional Setup Required
All examples use built-in Python libraries and the topological sort implementation.
