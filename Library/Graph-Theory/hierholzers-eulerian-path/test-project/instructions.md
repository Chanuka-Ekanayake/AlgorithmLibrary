## How to Run

No external dependencies — pure Python standard library.

### From the `hierholzers-eulerian-path` root directory:

```bash
cd test-project
python app.py
```

### Or from the repo root:

```bash
python Library/Graph-Theory/hierholzers-eulerian-path/test-project/app.py
```

### Expected Output:
Five demo sections covering:
1. Delivery van route optimizer (directed)
2. Königsberg Bridge Problem — the original Euler puzzle
3. Network cable inspection (undirected)
4. DNA sequence assembly via De Bruijn graph
5. One-stroke pen-drawing puzzle
