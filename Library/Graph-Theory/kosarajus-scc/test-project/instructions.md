# User Guide: Software Dependency Cluster Analyzer (Kosaraju's Algorithm)

This project simulates the backend health-check of an e-commerce platform that sells complex software bundles. It utilizes **Kosaraju's Algorithm** to scan the entire dependency catalog and instantly identify catastrophic circular dependencies (Strongly Connected Components).

## How to Test

1. **Navigate** to the `test-project` folder.
2. **Run** the simulator:
   ```bash
   python app.py
   ```
