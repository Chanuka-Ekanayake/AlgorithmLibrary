# User Guide: Real-Time Sales Analytics (Fenwick Tree)

This project simulates the backend of a live financial dashboard using a **Fenwick Tree (Binary Indexed Tree)**. It demonstrates the ability to ingest live, high-frequency data streams while instantly serving cumulative metrics.

## How to Test

1. **Navigate** to the `test-project` folder.
2. **Run** the simulator:
   ```bash
   python app.py
   ```
