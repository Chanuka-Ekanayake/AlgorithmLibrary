# User Guide: Software License Key Encryptor (Blowfish)

This project simulates a secure backend transmission for distributing digital assets (like software licenses or ML model access tokens). It uses the **Blowfish** symmetric block cipher to encrypt the payload before transit.

## How to Test

1. **Navigate** to the `test-project` folder.
2. **Run** the simulator:
   ```bash
   python app.py
   ```
