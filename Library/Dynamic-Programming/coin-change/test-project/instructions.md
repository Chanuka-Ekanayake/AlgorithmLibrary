# User Guide: API Credit Bundle Allocation (Coin Change)

This project simulates a resource allocation backend for an e-commerce platform. It contrasts a standard "Greedy" algorithm against a **Dynamic Programming** engine to fulfill a specific user request for computational API credits.

## How to Test

1. **Navigate** to the `test-project` folder.
2. **Run** the simulator:
   ```bash
   python app.py
   ```
