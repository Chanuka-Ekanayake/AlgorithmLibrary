# User Guide: Stock Trend Analyzer (LIS)

This project demonstrates the **Longest Increasing Subsequence** algorithm in a real-world data mining context. It evaluates a volatile data array to find the longest strictly increasing sequence hidden within the "noise."

## How to Test

1. **Navigate** to the `test-project` folder.
2. **Run** the simulator:
   ```bash
   python app.py
   ```
