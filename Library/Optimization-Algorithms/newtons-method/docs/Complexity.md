## 1. Single Variable Systems (1D)

The single-variable root finding and optimization pipelines strictly operate on univariate inputs where matrices and heavy memory allocations are entirely avoided.

### 1.1 Time Complexity

The time complexity of Newton's Method depends entirely on two factors:
1. The cost of evaluating the function and its derivative.
2. The total number of iterative steps required to reach the target tolerance.

Let $C$ represent the time required to evaluate the explicit function $f(x)$ and its derivative $f'(x)$.
Let $K$ represent the total number of iterations executed.

- **Analytical Derivative Injection**:
If the user explicitly supplies the analytical derivative as a programmatic lambda or function, the cost per iteration is precisely one evaluation of $f(x)$ and one evaluation of $f'(x)$. Thus, the time per step is $O(C)$.

- **Numerical Derivative Fallback**:
If the derivative is completely unknown and numerical approximation (finite differences) is used, the system essentially evaluates the main function $f(x)$ structurally twice closely around the coordinate $x_n$ (i.e., $f(x+h)$ and $f(x-h)$). The cost per step remains proportional to evaluating the main function, so it still equates to $O(C)$.

**Convergence Factor ($K$)**:
Under ideal conditions, Newton's Method achieves quadratic convergence. This means the number of correct decimal places theoretically doubles with every iteration. A high-precision result can typically be achieved within 5 to 10 iterations ($K \approx 10$), which for all practical purposes behaves as an algorithmic constant.

**Overall Time Complexity**: $O(K \cdot C) \to \mathbf{O(C)}$ assuming $K$ is small and bounded.

### 1.2 Space Complexity

The space complexity for single variable optimization is remarkably minimal. The algorithm requires maintaining no historical arrays, matrices, or structural lookup tables.
- Variables logically tracked include the current coordinate $x_n$, the next coordinate $x_{n+1}$, evaluations of $f(x_n)$ and $f'(x_n)$, and basic loop counters.
- The use of purely iterative loops prevents recursion depth stacking internally.

**Overall Space Complexity**: $\mathbf{O(1)}$
