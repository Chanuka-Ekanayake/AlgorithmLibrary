# User Guide: Linear Regression Trainer (Gradient Descent)

This project simulates the training phase of a Machine Learning model. It uses **Batch Gradient Descent** to optimize the weights and bias of a linear regression model by minimizing the Mean Squared Error.

## How to Test

1. **Navigate** to the `test-project` folder.
2. **Run** the simulator:
   ```bash
   python app.py
   ```
