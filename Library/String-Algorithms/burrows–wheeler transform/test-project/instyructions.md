# User Guide: Software Binary Pre-Compressor (BWT)

This project simulates the preprocessing pipeline for data compression using the **Burrows-Wheeler Transform**. It demonstrates how scrambling a string mathematically reduces its sequence variance, preparing it for high-ratio compression algorithms.

## How to Test

1. **Navigate** to the `test-project` folder.
2. **Run** the simulator:
   ```bash
   python app.py
   ```
