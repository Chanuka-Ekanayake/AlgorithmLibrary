# Suffix Tree - Running the Test Project

## 1. Prerequisites

- **Python 3.x** must be installed on your system.

---

## 2. Instructions

To execute the test suite and verify the implementation of Ukkonen's Suffix Tree:

1.  Open your terminal/command prompt.
2.  Navigate to the `test-project` directory:
    ```bash
    cd "Library/String-Algorithms/suffix-tree-ukkonen/test-project"
    ```
3.  Run the test script:
    ```bash
    python app.py
    ```

---

## 3. What the Test Script Does

- **Comprehensive Pattern Matching**: Runs multiple test cases with various string lengths and patterns (e.g., "BANANA", "MISSISSIPPI").
- **Verification**: Formats output with `[PASSED]` or `[FAILED]` based on expected status.
- **DNA Sequence Demo**: Simulates a basic bioinformatics search within a DNA sequence to demonstrate the efficiency on structured repetitive strings.
