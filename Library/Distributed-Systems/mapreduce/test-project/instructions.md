# User Guide: E-commerce Log Analyzer (MapReduce)

This project demonstrates a **MapReduce** pipeline processing 20,000 simulated server logs. It uses Python's `ProcessPoolExecutor` to spin up parallel worker processes, simulating a distributed cluster environment.

## How to Test

1. **Navigate** to the `test-project` folder.
2. **Run** the simulator:
   ```bash
   python app.py
   ```
